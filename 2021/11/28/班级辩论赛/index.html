# 班级辩论赛

# “谢谢主席，问候在场各位

        “我方观点是：不应该淡化奥运会金牌情结……” 再焦急和忐忑的等待，都化作赛场上的侃侃而谈。数万字的材料、三番五次修改的辩稿，都化作赛场上凌厉的诘问和从容的对答。终场的铃声响起，主席的话音落下，我们再一次为来之不易的胜利而欢呼。

        明黄色的秋风悄然掠起，又是辩论好时节。准备的过程中，真的有太多太多的温暖和感动。在第一套思路行不通的时候，所有人围在一起，眼中净是迷茫；但当我们苦思冥想，脑汁绞尽，终于出炉第二套思路的时候，已经流下了泪水。多少个月朗星稀的夜晚，教学楼的寒灯独夜，队员们促膝深谈，在通往宿舍的静谧校道上一边奔跑，一边让思维的火花肆意碰撞，然后在一天的劳累后不觉困意，这大概就是最好的睡前故事吧。

        周一下午的通用电教室，座无虚席。同学们提着晚餐赶来，兴致高涨，掌声雷鸣。对方辩友抛出崭新的观点，大家和队员一起凝眉沉思；我方队员发出巧妙的反驳，大家也不禁笑逐颜开。得益于充分的准备，我方队员兵来将挡，水来土掩，伶牙俐齿，慷慨陈词，在班徽的映照下充分展现了卓越的风采！

        复赛结束，半决赛在即。感谢金城的包子，在飘香中将永远留下我们一起奋斗的回忆。

# 绿茵场上 又有你的身影

# 记第二次足球赛

  玻璃一般湛蓝的球衣，穿梭在盛夏的绿茵之中，环绕以同样色调的蓝白球迷，煞是好看。烈日将跑道灼烧得直吐热气，人像种子种在这样的土地上，不一会儿就晒脱了水。可是，没有什么能够阻挡运动员们肆意驰骋与飞奔，还有永不言败的英雄的心。那是向着胜利熊熊燃烧的一把火焰。

  看吧，皮球在队员间眼花缭乱地默契传切，晃动，变向，闪出空当，便似撒蹄骏马，如利刃尖刀，对准禁区的心脏猛插过去。屡屡给对手以如此犀利的进攻，带动着多少观众紧张的心，和对面的防线一起颤动着；一记打门惊险地遭到扑出与解围，又给我们带来多少的遗憾。在一串串连续的威胁中，身为观众的笔者都感到神经经受不住这样的紧绷，可想而知，场边站立的层层观众早已热血沸腾。

  而当险情发生在己方的门前，我的心更是跳到嗓子眼了。尚不及回防的前锋焦急地祈祷后卫能完成一次漂亮的防守，而接到长传的对方前锋又怎么甘心白白丧失机会？他一马当先，长驱直入，可是我们的防守者注定专为大场面而生：只见紧随其后的队员沉着冷静，在抽射前千钧一发的刹那，一招精准的滑铲，将迫在眉睫的危机化于无形，空留懊悔万分的进攻手久久叹息。而它受到的叫好之声，也已经无法用激动二字形容了。

  终场哨毫无悬念地响起，看着垂头丧气的敌手，又血拼出一场胜利的全队队员与冲上球场的观众又可以一起庆祝了。

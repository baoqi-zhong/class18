# 九月份新青年

# 陈俊宇

当你一脸严肃地回答问题时，你是正儿八经的好学生；当你课间见缝插针地切磋交流奥数时，你是兴趣引领着的数学英才。一记三分的失准不会影响你的下一次自信出手，正如你平日里笑对 “zei 分” 时沧海轻舟的豁然开朗。一号，不仅是姓氏的排列，还意味着开拓、挑战，更呼唤着新的青年！

# 宋亦非

一只粉笔，一个礼拜，一个奇迹。细笔勾勒，美少女的神韵渐见，凝视她的瞳孔，让人不愿相信美少女竟身处二维平面。感谢你的用心，挤占自习与休息时间，换来这一份理想的美好的诞生，更感谢，你对漫画的专注投入，诠释了一位新青年的深意内涵。

# 唐宏凯

一双手，可以印刷出令人咋舌的英文字母，也可以开出刺激痛感神经的旋球；一张嘴，可以友善亲热地开着无厘头的玩笑，也可以作出 "very good~~" 的英语发言。一颗心，在静如处子，动若脱兔的两种境界游刃有余，也真诚炽热地对待班上的每一份子。一位青年，何以谓之新？常新地活在每个人的心里，是以谓之新也。

# 谢砚琪

倘若少了你，繁杂的课程就会失去它的秩序；倘若少了你，图书角就会失去它的温馨的装饰而顿失吸引力。你如夜空中微光闪烁的星体，将永恒的光辉穿透茫茫的宇宙。事虽细小，贵在持之以恒。你又如晴空下划过的流星，擦生出不期而至的光亮，娃娃的范文，经的起教鞭的评点。你是班上为数甚少的 “稀有物种”，珍惜着一份可贵的美好

# 曾子晴

该怎样形容你的美丽？你一笔一划创作板报的模样，仿佛一座圣洁静穆的雕像，于无声中诉说着十八班人的底蕴；你那一抹微笑绽放的善意，似雨后纯净的花的清香，芬芳了校园的时光。你以铭记责任为乐，将部长的重任置于肩上；你总在默默之处燃烧小我，留给师生一个明亮温馨的家。千言万语，诉不尽你的美丽！

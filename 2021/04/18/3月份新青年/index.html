# 三月份新青年颁奖

金牌主持人 雷颜玮

# 韦睿行

  群鸿戏海，舞鹤游天。你信手挥墨，观之若脱缰野马腾空而来又绝尘而去。三步上篮，精准罚球。出于热爱，你的球技炉火纯青;出于**责任**，你出尽全力为18班镌刻一段独特时光。你是**文雅的书生**，你是**球场上的光**。

# 钟梓洋

  轻敲键盘，微抚鼠标，IT尖兵目不窥园，历尽心血。将班级门牌挂在**互联网**的街道，把卓越的牌面树在一中人眼前。错过回家的班车，你是默默**奉献**的，一心为班的新青年。

# 梁俊文

  执一把扫帚，迎一抹残阳，**书一份责任**。你无怨无悔，将班级道义担在肩上，将卫生的**整洁**记在心里。

# 李可

  配上一颗闪闪的**团徽**，翻开，一页厚厚的历史。你潜心贯注，心会神凝，深宵的灯火化作湘江的激昂，为我们传承**红色的基因**。

# 叶宇桐

  一个焦虑的背影，匆匆从走廊这头走到那头，从二楼爬上七楼，他只为确保每一个值周岗位万无一失;一个**真切**的声音，有时殷勤叮嘱，有时不怒自威，无论何时何地，他心里总**惦记着班级**。他就是班级的主心骨，当之无愧的领袖。

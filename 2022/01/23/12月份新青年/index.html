# 十二月份新青年

# 李可

        奋笔书，你在辩论赛上慷慨陈词；洒汗水，你在篮球场里奔跑跳跃。一腔真诚，一心尽职，一次次精彩纷呈的团课，一节节有序的早读，你守住前人的火焰，点亮同学们的一天。从文到理，由古至今，你饱读诗书，落笔惊风雨；满腹经纶，成文泣鬼神。不禁让人赞叹 “该生优秀，还会打球！”

# 邱童

        三尺讲坛是 TED 的舞台，英语沙龙，你侃侃而谈；一张便签是作业的清单，语文生物，你责无旁贷。

# 詹岚

        晨曦灿烂中，是你阅读的身影；余晖烁烁里，是你专注的神情。温柔恬静，低调热心，你从不炫耀自己的光芒，却将温暖洒向班级。从不懈怠，认真负责，收起的堆书不仅是责任的证明，更垒起你走向新青年的阶梯！轻拢一缕云岚，化为一片詹青。

# 周熙临

        口若悬河，是滚滚春潮不可当，乱石穿空，惊涛拍岸；齿芒于刃，是独孤九剑斩荆棘，天下无坚，击之必破。

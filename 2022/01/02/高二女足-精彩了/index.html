# 女足 ——“精彩了！”

精彩了！—— 重要蒋话

在不算寒冷的秋冬，我们迎来了高中唯一一届女足比赛，伴随着 ** 蒋麟的 “精彩了！”** 和体委祉舆的动员，最终熙临、思薇和我（楚清）成为了女足联队的球员。这是一段难忘的青春回忆，仍不能忘记一些场景。

# ①irst

1st 当属 “梅熙” 入球。疾影穿梭，步下生莲，绿茵场上，伴随着加油声和呐喊声，一粒快到试图逃脱视线的球射入球门，球身仍在快速旋转，妄图冲破球网。一旁比女生还激动的男同学们欢呼雀跃，比看男足还兴奋，在这场与五班的季军赛中甚至比哪方的 **“猿声” 高昂 **。

# ②econd

2nd 我们几次赛前准备。我们基本在比赛以外的时候没有配合过，但是没办法，我们有徐慧和 “梅熙”（骄傲脸），还有各位给力的队友：“C 罗” 罗思薇、文静、黄婷懿和最不容易、功劳最大的守门员等等。我们的交流和协商都非常顺利，马上就开始了热身和相互熟悉。

# ③hird

3rd 本人短暂的上场，真真切切体会到踢足球有多累，上场一分钟就开始口干舌燥，感觉整个人的水分流失严重，走位传球一个都不能松懈。每个上场的队员都很不容易，竭尽全力，只为更高的荣誉。

# ④ourth

3rd 源自那盏夜空下光芒四射的灯。每场比赛都会拖到大概六点半，而北半球的下半年总是昼短夜长。每每踢到最后，都会开盏大灯，这时候，仿佛这半个田径场自成一方天地，充盈着这个年纪永远的朝气蓬勃，强光使这片天地带着些许朦胧的味道，晕染着我们的少年时代。

当然，还要感谢各位同学、老师和家长的倾力支持和鼓励！

女足赛后更有悠长的余韵给予我们品味。我们在课下会更多地选择去享受绿茵场上奔跑和逐球的快感，我们更团结更齐心，我们也继续我们的旅程，相信前方，锦绣万里。

“我们是生生不熄的花火，终将以灿烂为名。”

文案：楚清
